# Plot magnetic moments of atoms

### How to use the code
- Clone the repository or download as ZIP
- Open 'plot_magmom.py'
- Set directory as the path of VASP outputs
- Run 'plot_magmom.py' with Python

### Customizing the plot
- Open 'plot_magmom.py'
- Edit the variables defined at the begining of the code

### Requirements
- Numpy
- Matplotlib
